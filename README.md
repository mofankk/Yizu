## Yizu（易租）

此项目为本人毕业设计项目，项目构思、代码设计编写以及论文编写均由本人完成。

### 项目结构描述

* api: 请求相关的参数校验以及返回值包装
* conf: 配置相关
* modules: 与数据库表结构映射以及其他数据结构
* router: 请求路由
* service: 逻辑处理与数据相关处理
* util: 工具箱
* img: README.md 引用图片
* server_config.json: 配置文件
